## 一、世界观设定

### 1.1 核心概念

**天道委员会**：修仙界最高行政机构，负责维护天地秩序、审核渡劫、分配资源、仲裁纠纷。

**核心理念**：修仙不是打打杀杀，是人情世故。

### 1.2 修仙界行政体系

```
天道委员会（中央）
├── 渡劫审批司
├── 功德考核司
├── 资源分配司
├── 纠纷仲裁司
├── 仙籍管理司
├── 功法审核司
└── 监察司

地方机构：
├── 东天门（管辖东部诸州）
├── 西天门（管辖西部诸州）
├── 南天门（管辖南部诸州）
├── 北天门（管辖北部诸州）
└── 中天门（直辖）

宗门关系：
- 宗门需在天道委员会注册
- 宗门晋升需考核
- 宗门违规会被处罚（罚款、降级、取缔）
```

### 1.4 境界体系

```
凡人 → 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 地仙 → 天仙 → 金仙 → 大罗金仙 → 仙尊 → 仙帝
```

### 1.5 功德体系

**功德**：修仙界的"信用分"，影响渡劫难度、资源分配、职位晋升。

**获取方式**：
- 完成天道任务
- 救助凡人
- 维护秩序
- 创新贡献

**扣除方式**：
- 违规施法
- 伤害凡人
- 破坏秩序
- 贪污腐败

**功德等级**：
- 负功德（黑名单）
- 0-1000（普通）
- 1000-5000（良好）
- 5000-10000（优秀）
- 10000+（模范）

## 二、天道委员会详细设定

### 2.1 渡劫审批司

**职能**：审核渡劫申请、安排渡劫场地、现场监考、记录结果。

**部门架构**：
- 司长（正三品）
- 渡劫审批处（正四品）
  - 材料审核科（正五品）
  - 场地安排科（正五品）
- 现场监考处（正四品）
  - 天劫记录科（正五品）
  - 应急救援科（正五品）

**工作流程**：
1. 申请人提交材料（修为证明、功德证明、担保函）
2. 材料审核（3-5 个工作日）
3. 安排渡劫场地（根据天劫等级）
4. 现场监考（防止作弊、记录数据）
5. 出具渡劫证明（成功/失败）

**常见猫腻**：
- 加急费（正常 3 天，加急 3 小时）
- 场地费（好场地天劫弱 10%）
- 人情劫（监考时放水）

### 2.2 功德考核司

**职能**：核算修士功德、发放功德补贴、处理功德异议。

**考核周期**：
- 小考：每年一次
- 大考：每百年一次

**考核内容**：
- 善行记录
- 违规记录
- 贡献值
- 群众评价

### 2.3 资源分配司

**职能**：分配灵气资源、灵脉管理、丹药配额、法宝指标。

**核心资源**：
- 灵脉开采权
- 丹药炼制配额
- 法宝炼制指标
- 秘境进入名额

### 2.4 纠纷仲裁司

**职能**：仲裁宗门纠纷、资源争夺、师徒矛盾、道侣分割。

**仲裁原则**：
- 依法裁决（天道法典）
- 兼顾人情
- 维护稳定

### 2.5 仙籍管理司

**职能**：管理修仙户籍、修为登记、出入境管理、黑名单管理。

**仙籍类型**：
- 正式仙籍（有编制）
- 临时仙籍（暂住证）
- 黑仙籍（无身份）

### 2.6 功法审核司

**职能**：审核新功法、鉴定功法等级、打击盗版功法。

**功法等级**：
- 天阶（禁传）
- 地阶（限制传播）
- 玄阶（需备案）
- 黄阶（自由传播）

### 2.7 监察司

**职能**：监督各部门、查处腐败、处理举报、内部审计。

**特权**：
- 可直接逮捕任何品级官员（需证据）
- 可调阅任何档案
- 可直接向委员长汇报

## 三、核心规则设定

### 3.1 渡劫流程

```
1. 提交申请（填写《渡劫申请表》）
2. 提交材料：
   - 修为证明（元婴/金丹等）
   - 功德证明（功德≥1000）
   - 担保函（至少一位金仙担保）
   - 体检报告（神魂稳定）
3. 缴纳费用：
   - 审核费：100 灵石
   - 场地费：500-5000 灵石（根据天劫等级）
   - 保险费：200 灵石（可选，失败可赔）
4. 等待审核（3-5 工作日）
5. 安排渡劫（通知时间地点）
6. 现场监考（至少两名科员）
7. 出具证明（渡劫成功/失败）
```

### 3.2 晋升流程

```
1. 满足条件（修为、工龄、功德）
2. 提交申请（《晋升申请表》）
3. 参加考核（笔试 + 面试 + 实操）
4. 公示（7 天）
5. 任命（颁发官印）
```

### 3.3 考核制度

**年度考核**：
- 德（品德）
- 能（能力）
- 勤（出勤）
- 绩（业绩）
- 廉（廉洁）

**考核结果**：
- 优秀（晋升优先）
- 称职（正常）
- 基本称职（警告）
- 不称职（降职/辞退）

### 3.4 休假制度

- 年假：10 天/年（工龄每增 1 年 +1 天，最多 30 天）
- 事假：需审批，扣功德
- 病假：需医院证明
- 产假：女仙 90 天，男仙 15 天
- 渡劫假：申请渡劫可请 7 天假

## 六、伏笔设计

### 6.1 主线伏笔

1. **主角身份**：穿越者，前世是公务员（埋线：为什么熟悉体制）
2. **天道委员会秘密**：表面是行政机构，实际在策划大阴谋
3. **功德体系真相**：功德不是善行，是某种能量
4. **天劫真相**：天劫不是考验，是收割
5. **最终 BOSS**：天道委员长不是好人

### 6.2 配角伏笔

1. **上司**：表面贪财，实际在收集证据查大案
2. **同事**：表面老实，实际是关系户
3. **红颜**：表面是凡人，实际是大佬转世
4. **对手**：表面是反派，实际是被冤枉的

### 6.3 世界观伏笔

1. **修仙界真相**：整个世界是个"养殖场"
2. **飞升真相**：飞升不是解脱，是成为"收割者"
3. **功德真相**：功德是"货币"，天道是"银行"
4. **历史真相**：上古时期有过反抗，失败了
